# trabaho-principios-de-construcao-de-algoritmos

Nomes: Karina Lussac e Aramis Neto

nesse repositório, nós temos um sistema em Python para cadastrar alunos usando CSV e pandas. 
Dá pra inserir, pesquisar, editar e remover os dados dos alunos, a matrícula é gerada automaticamente. 
O menu fica em loop até o usuário escolher sair.
Para usar: instalar pandas (pip install pandas) e rodar o arquivo main.py no terminal.
Commits feitos de forma contínua, sempre descrevendo as mudanças, como criação de funções, ajustes na busca, edição e remoção, e correções no código.
